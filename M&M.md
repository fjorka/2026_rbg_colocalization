To quantify whole-cell expression levels and spatial co-distribution, multi-channel wide-field fluorescence images comprising DAPI, Slc39a9 (Cy3), and a lipid raft marker (FITC) were processed using a custom Python pipeline. Prior to segmentation, pixel intensities for each channel were normalized by clipping values between the 3rd and 99.9th percentiles. Whole-cell body segmentation was performed on the normalized multi-channel stacks using the Cellpose deep learning framework (v4.1.1; Stringer et al., 2021). Spurious segmentation artifacts and incomplete cells were filtered using scikit-image (v0.26.0; van der Walt et al., 2014); specifically, objects with a total area smaller than 400 pixels were removed, and cells touching the image borders were cleared from subsequent analysis.
Image-wide background values for each channel were determined by calculating the median fluorescence intensity of the unsegmented background regions (where cell mask == 0). These background values were subtracted from the raw pixel intensities of each channel prior to downstream quantification. Whole-cell expression was calculated as the mean intensity per segmented cell mask. To evaluate the local spatial relationship between Slc39a9 and lipid rafts marker a per-pixel Spearman rank correlation coefficient was calculated within each individual single-cell mask using scipy.stats (v1.17.1; Virtanen et al., 2020).
